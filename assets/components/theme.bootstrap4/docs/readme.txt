theme.Bootstrap4 — базовая тема на фреймворке Bootstrap 4. Текущая версия v4.0.0-alpha.

<b>Префиксы</b>
- шаблонов — tpl.bt4
- чанков — ch.bt4

<b>Используемые чанки:</b>
- ch.bt4.breadcrumb.tpl - Чанк с хлебными крошками <a href="http://v4-alpha.getbootstrap.com/components/breadcrumb/" title="Breadcrumb" target="_blanc">Breadcrumb</a>
- ch.bt4.card.tpl - Чанк с карточками для отображения контента <a href="http://v4-alpha.getbootstrap.com/components/card/" title="Card" target="_blanc">Card</a>
- ch.bt4.carousel.tpl - Чанк со слайдером <a href="http://v4-alpha.getbootstrap.com/components/carousel/" title="carousel" target="_blanc">Сarousel</a>
- ch.bt4.footer.tpl - Чанк с футером
- ch.bt4.head.tpl - HEAD
- ch.bt4.jumbotron.tpl - Чанк с Jumbotron. Компонент Jumbotron предназначен для создания контента или информации на веб-сайте, который занимает всю ширину контейнера куда он помещен и является очень большим, чтобы пользователи обратили на него внимание.<a href="http://v4-alpha.getbootstrap.com/components/jumbotron/" title="Jumbotron" target="_blanc">Jumbotron</a>
- ch.bt4.navbar.tpl - Чанк с навигационным меню <a href="http://v4-alpha.getbootstrap.com/components/navbar/" title="Navbar" target="_blanc">Navbar</a>

<b>Шаблоны</b>
tp.bt4.onepage.tpl - New in Bootstrap 4 Theme (http://www.codeply.com/render/kNmg5E60WS)
tp.bt4.home.tpl - Шаблон с демонстрацией содержимого чанков

Как использовать всё можно найти на v4-alpha.getbootstrap.com, есть примеры.
Галерею шаблонов можно посмотреть тут: <a href="http://themes.getbootstrap.com/" target="_blank" title="Themes">Getbootstrap</a>
В качестве примера, в некоторых чанках (навигация, хлебные крошки), использован pdoTools. Его использование не обязательно.

Официальный сайт фреймворка:  <a href="http://v4-alpha.getbootstrap.com/" title="Bootstrap 4" target="_blanc">Bootstrap 4</a>


A basic theme for MODx Revolution based on Twitter Bootstrap 4 (http://v4-alpha.getbootstrap.com/) + New in Bootstrap 4 Theme (http://www.codeply.com/render/kNmg5E60WS)

You can change your default template: just add the name of the folder where others template are, for example:
instead of default bootstrap template:
<link href="[[++assets_url]]components/theme.bootstrap4/css/bootstrap.min.css" rel="stylesheet">

Integrated with pdoTools. You do not need to install any additional snippets for basic functionality.
